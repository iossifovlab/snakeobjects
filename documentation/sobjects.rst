.. _sobjects-commands:

``sobjects`` commands
------------------------------

This is the main ``snakeobjects``' command line tools. 
The general usage has the ``sobjects <command> [<option 1> <option 2> ...]``.
The available commands are described below:

.. program:: sobjects 


.. option:: version
.. runblock:: console

    $ sobjects help version


.. option:: help 
.. runblock:: console

    $ sobjects help help 


.. option:: describe
.. runblock:: console

    $ sobjects help describe 

.. option:: buildObjectGraph
.. runblock:: console

    $ sobjects help buildObjectGraph

.. option:: createSnakefile
.. runblock:: console

    $ sobjects help createSnakefile

.. option:: createSymbolicLinks
.. runblock:: console

    $ sobjects help createSymbolicLinks


.. option:: prepare
.. runblock:: console

    $ sobjects help prepare


.. option:: prepareTest
.. runblock:: console

    $ sobjects help prepareTest

.. option:: run 
.. runblock:: console

    $ sobjects help run 

.. option:: cleanProject
.. runblock:: console

    $ sobjects help cleanProject

.. option:: submit 
.. runblock:: console

    $ sobjects help submit 

.. option:: graph
.. runblock:: console

    $ sobjects help graph
